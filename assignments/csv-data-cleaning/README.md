# 📘 Assignment: CSV Data Cleaning Challenge

## 🎯 Objective

Practice cleaning messy CSV data in Python by standardizing values, handling missing information, and producing a simple summary report.

## 📝 Tasks

### 🛠️ Load and Inspect the Data

#### Description
Load a provided CSV file and inspect its structure before cleaning it.

#### Requirements
Completed program should:

- Read a CSV file using Python
- Print the number of rows and columns
- Display the first few rows to inspect the data

### 🛠️ Clean the Data

#### Description
Fix inconsistent and messy values so the dataset is easier to analyze.

#### Requirements
Completed program should:

- Remove extra whitespace from text values
- Standardize capitalization for names or categories
- Replace empty values with a clear placeholder or skip them safely
- Convert numeric fields to integers or floats when possible

### 🛠️ Create a Summary Report

#### Description
Turn the cleaned data into a short report that highlights useful information.

#### Requirements
Completed program should:

- Save the cleaned data to a new CSV file
- Print a simple summary such as the number of valid rows or average values
- Include at least one insight from the cleaned dataset
